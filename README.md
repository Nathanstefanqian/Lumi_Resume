# Lumi Resume - 智能简历编辑器

Lumi Resume 是一款专为开发者和学生打造的高效率、响应式、且集成 AI 能力的在线简历编辑平台。它不仅提供了精美的 A4 尺寸实时预览，还深度集成了 DeepSeek 模型，帮助用户将平淡的描述转化为专业的职场语言。

## ✨ 核心特性

- **🚀 实时 A4 预览**：基于浏览器的分页算法，确保“所见即所得”，完美适配 A4 打印标准。
- **🤖 AI 润色助手**：集成 DeepSeek-V3 模型，一键将模糊的自我评价润色为专业的简历要点。
- **📸 智能照片上传**：支持头像直传阿里云 OSS，具备预签名安全机制。
- **🗺️ 智能字段编辑器**：
  - **年龄计算器**：输入生日自动换算为年龄。
  - **地区级联选择**：规范化的省市联动输入。
  - **日期范围选择**：优雅的在校/工作时间区间选择器。
- **📁 简历库管理**：支持多份简历管理、克隆、另存为以及历史版本恢复。
- **🎨 极致交互体验**：毛玻璃加载动效、顶部 Toast 提示系统、自定义确认模态框，告别原生丑陋弹窗。
- **📤 多格式导出**：支持一键导出高质量 PDF 和 PNG 图片，文件名根据姓名动态生成。

## 🛠️ 技术栈

- **前端**：React 19, Vite, Tailwind CSS 4, Lucide Icons
- **后端集成**：对接 Lumi-be (NestJS)，支持 JWT 认证与 OSS 存储
- **导出技术**：html-to-image, Puppeteer (服务端导出)

## 📦 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 构建生产版本
```bash
npm run build
```

## 🤝 关联项目
本项目作为 Lumi 生态的一部分，需配合 [Lumi-be](https://github.com/Nathanstefanqian/Lumi-be) 后端运行以获得完整的 AI 与存储能力。
